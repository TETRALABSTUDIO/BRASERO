Drop your carousel slide exports here (1080 x 1350 px, JPG or PNG).

Each "Deck" card in the Collection is itself a mini carousel you can swipe
with the arrows. Name files as  DECK-SLIDE.jpg :

HERO (fanned stack, front to back):
  hero-1.jpg   <- front / featured
  hero-2.jpg
  hero-3.jpg

COLLECTION  (deck number - slide number):
  1-1.jpg  1-2.jpg  1-3.jpg     <- deck 1, slides 1/2/3
  2-1.jpg  2-2.jpg  2-3.jpg     <- deck 2
  3-1.jpg  ...                   <- and so on (6 decks, 3 slides each)

Add more slides per deck by continuing the numbering (1-4.jpg, 1-5.jpg...).
Missing files fall back to a colored placeholder, so nothing ever looks broken.
Replace any time, no code changes needed.
